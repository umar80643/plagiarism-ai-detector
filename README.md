# 🤖 AI Plagiarism Detector

An AI-powered plagiarism detection system that analyzes text documents and measures similarity using Natural Language Processing (NLP) and Machine Learning techniques. This project aims to provide an efficient way to identify copied or highly similar content.

---

## 🚀 Features

- Compare text documents for similarity
- Generate plagiarism percentage
- Text preprocessing using NLP techniques
- Fast similarity analysis
- Simple and user-friendly interface

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Streamlit

---

## 📌 How It Works

1. Upload or enter text.
2. Preprocess and clean the text.
3. Convert text into numerical representations.
4. Calculate similarity using NLP techniques.
5. Display the plagiarism score.
